# Grid-Based Clustering Using Boundary Detection (GCBD) – Python Implementation

## Introduction

This repository provides a Python implementation of the **Grid-Based Clustering Using Boundary Detection (GCBD)** algorithm proposed by:

> Mingjing Du, Fuyu Wu. *Grid-Based Clustering Using Boundary Detection*. Entropy, 2022, 24(11): 1606.

The original algorithm was released by the authors in MATLAB. This project reimplements the method in Python while preserving the core methodology and experimental workflow described in the paper.

The implementation is intended for research, educational purposes, and reproducible experiments on grid-based clustering techniques.

## Reference

The original paper is available at:

Du, M., & Wu, F. (2022). *Grid-Based Clustering Using Boundary Detection*. Entropy, 24(11), 1606.

## Requirements

* Python 3.9+
* NumPy
* SciPy
* Pandas (optional)
* Scikit-learn (for evaluation and preprocessing)

Install dependencies:

```bash
pip install numpy scipy pandas scikit-learn
```
